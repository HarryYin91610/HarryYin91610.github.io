效果展示

理解css 3d变换

  • css 3d变换包含rotateX, rotateY, rotateZ，它们分别表示元素绕x、y、z轴进行旋转；
  • 光有3d变换还无法实现3d透视效果，这时perspective属性就必不可少，正如其名，这个属性的作用是设置视点和透视元素之间的距离（以像素为单位），默认视点的位置是在透视元素的正中心且距离为0。

  • 通俗一点，所谓视点就是从哪个位置去看这个透视元素，如下图所示：



  • 通过perspective-origin属性可以改变视点在透视元素的位置，默认是其正中心。
